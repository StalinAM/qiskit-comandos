# Comandos para emulador qiskit

## Instalacion

- Descargar [Anaconda](https://www.anaconda.com/download)
- Abrir Anaconda Prompt
- `conda create --name cwq`
- `conda activate cwq`
- `conda install pip`
- `pip install qiskit`
- `pip install qiskit-ibm-runtime`
- `pip install qiskit[visualization]`
- `pip install jupyter`

- Abrir notebook
- `jupyter notebook`
